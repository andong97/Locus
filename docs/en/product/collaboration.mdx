Locus builds a relatively complete collaboration interface around Git, so writing changes, reviewing diffs, resolving conflicts, and committing results can stay in the same workspace.

## Current Collaboration Capabilities

- Repository detection and initialization
- Unity project `.gitignore` initialization
- History graph and branch view
- `staged / unstaged / conflict` workspaces
- Commit details, stash, and submodule panels
- Built-in Git terminal
- AI-assisted commit message
- File diff linked with Unity semantic diff

## Why This Collaboration Approach Matters

For Unity teams, the hardest things to review are often mixed changes to assets and scripts rather than pure code. Locus puts Git workflows and Unity semantic views in the same interface, which reduces context switching and review cost.
