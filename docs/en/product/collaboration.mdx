Locus offers a fairly complete collaboration UI built around Git, so writing changes, reviewing diffs, resolving conflicts, and shipping commits all stay in one workspace.

## Current Collaboration Capabilities

- Repository detection and initialization
- Unity-project `.gitignore` initialization
- History graph and branch view
- Staged / unstaged / conflict workspaces
- Commit details, stash, and submodule panels
- Built-in Git Terminal
- AI-assisted commit messages
- File diff and Unity semantic diff, integrated

## Why This Style Matters

For Unity teams, the hardest changes to review are usually not pure code—they're mixed changes across assets and scripts. Putting the Git workflow and Unity semantic views in the same place cuts both context switching and review cost.
