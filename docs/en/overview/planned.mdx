The next phase of Locus is shaped by four themes: interaction surface, deeper Unity capabilities, team data entry points, and agent extensibility.

## Interaction and Host

- A vertical layout mode for narrow windows and higher information density.
- An option to embed Locus inside a native Unity Editor window, so you don't have to switch back and forth.

## Deeper Unity Capabilities

- More Unity-domain knowledge Skills.
- Runtime analysis with frame-by-frame sampling.
- More parsing, search, and invocation surfaces exposed on the Unity Editor side.
- Runtime hot updates with shorter feedback loops.

## Team and Data

- More collaboration entry points, e.g. send/receive over instant messaging.
- More config-data editing surfaces, e.g. Excel-style data maintenance.
- More VCS support, e.g. Perforce (P4) and SVN.

## Agent and Extensions

- A stronger end-to-end mode and implementation verification path.
- More external tool imports: CLI, MCP, and a fuller Skill Hub.
