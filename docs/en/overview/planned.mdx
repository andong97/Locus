The next phase of Locus is mainly oriented around interaction shape, deeper Unity capabilities, team data entry points, and agent extensibility.

## Interaction and Host

- Support a vertical layout mode that fits narrow windows and higher information density
- Support embedding inside a native Unity Editor window to reduce switching between the application and the editor

## Deeper Unity Capabilities

- Add more Unity-domain knowledge Skill
- Add runtime analysis and frame-by-frame sampling capabilities
- Expose more parsing, search, and invocation capabilities on the Unity Editor side
- Support runtime hot updates with shorter feedback loops

## Team and Data Capabilities

- Support more collaboration entry points, such as instant messaging send and receive
- Support more configuration-data editing capabilities, such as Excel-like data maintenance
- Expand support for more version control software, such as P4 and SVN

## Agent and Extensions

- Strengthen end-to-end mode and implementation verification capability
- Support more external tool imports, such as CLI, MCP, and a more complete Skill Hub
