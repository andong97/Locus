Locus is better suited to real Unity projects. It brings conversations, editor state, asset references, team knowledge, and Git review into the same desktop workspace so the agent can continuously complete a full development chain.

## One Workspace Covers the Full Loop

- Describe requirements and constraints in the conversation
- Read project context such as scripts, Prefabs, Scenes, Materials, and ScriptableObjects
- Use the Unity plugin, asset index, and Git review capabilities
- Feed compile results, asset changes, and modification results back into the same conversation

## Three High-Frequency Scenarios

### Fix a Real Feature

Read the relevant scripts and assets, complete the changes, request Unity recompilation, and return the verification results directly to the conversation.

### Investigate Asset Problems

Work around the asset database, dependency graph, and risk report to locate `Missing Script`, broken references, duplicate GUIDs, and impact scope.

### Review Complex Changes

Put Git changes, file diff, and Unity semantic diff in one workspace so changes to Scene, Prefab, Material, and similar assets are easier to understand.
