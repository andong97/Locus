title: Roadmap
description: What's coming next
---

## In Progress

- **Vertical layout and native Unity-window embedding**: a vertical layout for narrow windows; an option to embed Locus inside a native Unity Editor window.
- **Runtime analysis tools**: the Agent writes conditional, frame-by-frame sampling scripts during play mode to help with debugging.

## In Queue

- **More Unity Skills**: knowledge and memory maintenance (`/dream`), particle systems, profiler-driven performance triage, custom render pipelines, and more.
- **Chat-app integration**: send and receive Locus conversations from messaging apps.
- **More data editors**: support for additional config formats, such as Excel.
- **Unity Editor extensions**: expose Locus's parser, fast search, and other tools inside the native Unity Editor.
- **More VCS support**: Perforce (P4) and SVN.
- **Tighter security controls**: more granular tool permissions, plus smaller models that audit tool-call safety.

## Under Discussion

- **Local model runtime**: an optional local LLM runtime inside Locus.
- **Hot reload during Play mode**: edit script logic at runtime and have it compile and apply live.
- **End-to-end mode**: editor screenshots, scripted test inputs, and isolated evaluation contexts so the model can verify what it built.
- **External tool import**: load external tools via MCP, CLI, and similar.
- **Game-dev Skill Hub**: a community-driven sharing platform for Skills.
