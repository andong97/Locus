## Changes

### Added

- Added Unity runtime profiler helpers for `unity_run_states`, including metric sampling, spike capture, profiler frame export, CSV output, and JSON summaries.
- Added a built-in profiler skill at `skill/builtin/profiler.md`.
- Added Unity project launch from the chat Unity status popover, with launching and waiting-for-connection states.
- Added Unity asset-aware icons for Markdown references, asset chips, file trees, and Git change lists.
- Added compact vector and quaternion display in Unity inspector diffs, including Euler/Quaternion switching.
- Added a display setting for merging Git tree status into colored file icons.

### Fixed

- File changes now default to tree view and show Locus-managed folder badges in staging and commit detail trees.
- Cancelled assistant runs preserve interrupted streamed text and thinking content when a partial response exists.
- Knowledge tools now require `.md` suffixes for document paths and directory paths for directory operations.
- Built-in skills now resolve correctly from nested `skill/builtin/` paths, including legacy app skill selections.
- Knowledge catalog storage source labels now separate project and app built-in documents.
- The main app theme now defaults to dark.
- Roslyn bundle generation now writes through an isolated output directory and cleans transient ILRepack artifacts.
- NSIS updates clean legacy root-level built-in skill files after the built-in skill move.

### Removed

- Removed the legacy `scope` field from knowledge catalog indexing and tool-facing document metadata.
- Removed root-level bundled skill file paths in favor of `skill/builtin/`.
