## Changes

### Added

- Added managed Windows Python and Git runtimes, with runtime selection controls in Settings.
- Added release installer variants for builds with bundled Python/Git and builds without bundled Python/Git.
- Added folder selection in Collab staged/unstaged tree views for batch stage and unstage workflows.

### Fixed

- Improved Unity asset previews with syntax highlighting for text previews and default root target selection for prefabs.
- Fixed app bootstrap and session restore paths so they keep the user's saved reasoning effort defaults.
- Redacted sensitive Claude hook headers from logs.
- Aligned Collab focus ring styling.

### Removed

- Removed auth debug logging.
