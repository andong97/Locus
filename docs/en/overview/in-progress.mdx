This page highlights the directions that are already in active implementation and ongoing product refinement, based on the current desktop UI, repository modules, and shipped documentation.

## Desktop Workspace Foundation

The main app already centers on six workspaces: Chat, Knowledge, Collaboration, Assets, Agent, and Settings. That gives the product a stable skeleton for continued expansion.

## Unity Execution Loop

Workspace selection, Unity plugin connection, asset scanning, editor-side execution, and recompilation flow are already present and form the most important implementation track.

## Knowledge and Retrieval

`Design / Memory / Skill / Reference` already live in one workspace, with ongoing work around import, retrieval, injection preview, and retrieval configuration.

## Git and Semantic Diff

Git graph, staging, conflict handling, file diff, and Unity semantic diff are already converging inside one collaboration workspace, which is a strong base for further review improvements.
