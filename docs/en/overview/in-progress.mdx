This page surfaces what's actively being built and refined, drawn from the current desktop UI, repository modules, and shipped docs.

## Desktop Workspace Foundation

The app is built around six workspaces—Chat, Knowledge, Collaboration, Assets, Agent, and Settings—giving the product a stable skeleton to keep extending.

## Unity Execution Loop

Workspace selection, Unity plugin connection, asset scanning, editor-side execution, and the recompile flow are all in place. This is the most important implementation track right now.

## Knowledge and Retrieval

`Design / Memory / Skill / Reference` already live in one workspace. Ongoing work covers import, retrieval, injection preview, and retrieval configuration.

## Git and Semantic Diff

Git graph, staging, conflict handling, file diff, and Unity semantic diff are converging inside a single collaboration workspace—a strong base for further review improvements.
