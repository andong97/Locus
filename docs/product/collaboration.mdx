Locus 围绕 Git 做了较完整的协作界面，让写改动、看差异、处理冲突和提交结果都能留在同一个工作台。

## 当前协作能力

- 仓库探测与初始化
- Unity 项目 `.gitignore` 初始化
- 历史图谱与分支视图
- staged / unstaged / conflict 工作区
- 提交详情、stash、submodule 面板
- 内置 Git Terminal
- AI 辅助 commit message
- 文件 diff 与 Unity 语义 diff 联动

## 这套协作方式的意义

对 Unity 团队来说，最难审查的往往不是纯代码，而是资源与脚本混合变更。Locus 把 Git 流程和 Unity 语义视图放在同一界面里，可以减少上下文切换和审查成本。

- 与 [价值与场景](/product/use-cases) 一起看，理解它在真实任务里的位置
