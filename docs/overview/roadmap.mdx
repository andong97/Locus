description: Locus 后续开发规划
---

## 正在实施

- **垂直布局模式与Unity窗口内嵌**：支持在窗口缩窄时前端变为垂直布局，以及内嵌在Unity编辑器原生窗口中
- **运行时分析工具**：agent支持在游戏运行时编写条件触发脚本以逐帧动态采样数据，增强调试能力

## 队列中

- **更多 Unity 领域知识 Skill**：维护项目知识库与记忆（/dream）、创建粒子系统、实际读入 profiler 数据并分析项目性能瓶颈、编写自定义渲染管线等
- **即时通讯软件**：支持通讯软件收发会话消息
- **更多数据编辑工具**：支持编辑excel表格等更多配置数据
- **Unity编辑器拓展**：向Unity侧暴露Locus的解析、搜索等工具，在Unity原生编辑器内使用高速搜索等能力
- **更多版本管理软件支持**：P4、SVN支持
- **安全性增强**：增加更多工具权限配置项，使用小模型审核工具调用安全性

## 讨论中

- **本地语言模型部署**：在Locus中提供可选的本地模型运行时
- **编辑器Play Mode热更新**：支持在运行时修改脚本逻辑并实时编译应用
- **端到端模式**：通过编辑器内截图、编写测试输入脚本、单独上下文评估，增强模型对实现功能的测试、评估能力
- **外部工具导入**：支持MCP、CLI等方式导入外部工具
- **游戏开发 Skill Hub**：由社区驱动的游戏开发 Skill 共享平台
