Locus 更适合用于真实的 Unity 项目。它把会话、编辑器状态、资产引用、团队知识和 Git 审查整合到同一个桌面工作台中，让 Agent 能够持续完成完整的开发链路。

## 一个工作台覆盖完整闭环

- 在会话中描述需求与约束
- 读取项目上下文，如脚本、Prefab、Scene、Material、ScriptableObject 等
- 使用 Unity 插件、资产索引和 Git 审查能力
- 将编译结果、资产变更和修改结果反馈回同一个会话

## 三类高频场景

### 修复真实功能

读取相关脚本与资产，完成修改，请求 Unity 重编译，并将验证结果直接返回到会话中。

### 排查资产问题

围绕资产数据库、依赖图谱和风险报告定位 `Missing Script`、失效引用、重复 GUID 以及影响范围。

### 审查复杂变更

把 Git 变更、文件 diff 和 Unity 语义 diff 放在同一个工作台中，使 Scene、Prefab、Material 等资产的修改更易于理解。
