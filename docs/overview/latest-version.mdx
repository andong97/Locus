## 变更列表

### 新增

- 新增 Windows 托管 Python 与 Git 运行时，并在设置中提供运行时选择。
- 新增带内置 Python/Git 与不带内置 Python/Git 的安装包构建变体。
- Collab staged/unstaged 树视图支持选择文件夹，用于批量 stage 和 unstage。

### 修复

- 改进 Unity 资产预览：文本预览支持语法高亮，Prefab 默认选中根对象。
- 修复应用启动与会话恢复路径中推理强度默认值被覆盖的问题。
- Claude hook 日志会脱敏敏感请求头。
- 对齐 Collab 焦点样式。
