Locus 下一阶段主要围绕交互形态、更深入的 Unity 能力、团队数据入口和 Agent 可扩展性展开。

## 交互与宿主

- 支持适合窄窗口与更高信息密度的垂直布局模式
- 支持嵌入到 Unity 编辑器原生窗口中，减少应用与编辑器之间的切换

## 更深入的 Unity 能力

- 增加更多 Unity 领域知识 Skill
- 增加运行时分析与逐帧采样能力
- 在 Unity 编辑器侧暴露更多解析、检索与调用能力
- 支持反馈循环更短的运行时热更新

## 团队与数据能力

- 支持更多协作入口，例如即时通讯收发
- 支持更多配置数据编辑能力，例如类 Excel 的数据维护
- 扩展更多版本管理软件支持，例如 P4 与 SVN

## Agent 与扩展

- 强化端到端模式与实现验证能力
- 支持更多外部工具导入，例如 CLI、MCP 以及更完整的 Skill Hub
